# Guest Extraction Patterns

Title signals: `with First Last`, `featuring First Last`, `interview with First Last`, `First Last - Topic`, `Ep 42 - First Last, Company`, `Topic w/ First Last`, `First Last of Company`.

Description signals: LinkedIn URL on the same line as a name, guest bio paragraphs, and chapters that introduce a guest by full name.

Use OCR when thumbnails carry names and titles are vague. Exclude known hosts before matching guests to LinkedIn.
