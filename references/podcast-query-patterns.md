# Podcast Discovery Query Patterns

Use a fan-out set per niche:
- `best [niche] podcasts founder owner interview`
- `[niche] podcast host LinkedIn`
- `site:youtube.com/@ [niche] podcast`
- `site:podcasts.apple.com [niche] podcast host`
- `site:spotify.com/show [niche] podcast host`
- `[niche] "podcast" "with" "CEO"`
- `[niche] association podcast interview`
- `[niche] conference podcast speakers`

Prioritize owner-operated shows for host outreach and interview-style shows for guest prospecting.
